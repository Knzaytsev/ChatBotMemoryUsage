# Использование памяти в генеративных диалоговых моделях

Репозиторий содержит файлы и скрипты, которые использовались во время проведения исследования. Структура проекта следующая:


```
├── msc                 <- Папка со скриптами, которые получают необходимые данные из датасета MSC
|   ├── data
|   |   ├── msc         <- Папка с диалоговыми датаестом MSC
|   |   ...
|   ├── data_reader.py  <- Скрипт, считывающий суммаризации из сессий MSC
|   ├── label_studio_loader.py    <- Скрипт преобразования данных в требумый формат для разметки в label-studio
|   └── msc_reader.py             <- Скрипт загрузки диалоговых данных из MSC
└── experiments                   <- Папка с ноутбуками, в которых производится подсчёт статистик, обучение моделей и эксперименты в целом
    ├── data                      <- Папка с изначальными, предобработанными и результирующими данными экспериментов
    ├── opt_with_context          <- Папка модели OPT, использующей контекст
    |   └── 0_epoch_lora          <- Папка с конфигурацией и весами LoRA
    ├── opt_without_context       <- Папка модели OPT, не использующей контекст
    |   └── 0_epoch_lora
    ├── vicuna_experiments        <- Папка с результатами экспериментов с моделью Vicuna
    |   ├── vicuna_context        <- Папка результатов экспериментов с Vicuna, учитывающей контекст
    |   ├── vicuna_without_context    <- Папка результатов экспериментов с Vicuna, не учитывающей контекст
    |   ...
    ├── AnnotationGathering.ipynb     <- Ноутбук со сбором размеченных ответов от диалоговой модели и оценкой её качества
    ├── CausalLM_Fine_tuning.ipynb    <- Ноутбук с дообучение каузальной языковой модели на диалоговых данных
    ├── Data Concat.ipynb             <- Ноутбук со сбором данных для дообучения диалоговой модели
    ├── Facts Classifier (Bert).ipynb <- Ноутбук для дообучения классификатора на основе BERT
    ├── Facts Classifier.ipynb        <- Ноутбук для дообучения классификатора на основе энкодера T5
    ├── Facts predicting.ipynb        <- Ноутбук для предсказания релевантности факта с помощью классификатора
    ├── Facts_collecting (Summary).ipynb  <- Ноутбук для сбора суммаризация с помощью Vicuna
    ├── Facts_collecting.ipynb            <- Ноутбук для сбора фактов с помощью Vicuna
    ├── TelegramBot.py                    <- Телеграм-бот для разметки ответов диалоговой модели
    └── statistics.ipynb                  <- Ноутбук для сбора статистик и рисования графиков
```
